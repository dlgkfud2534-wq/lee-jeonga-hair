## 3. 자간 (Letter Spacing)

| 요소 | 값 |
|---|---|
| 히어로 메인카피 (대형) | -0.02em ~ -0.03em (타이트) |
| 섹션 타이틀 | -0.01em ~ -0.02em |
| 서브 헤딩 | -0.01em ~ 0em |
| 본문 | 0em ~ 0.01em (기본) |
| 캡션/소형 텍스트 | 0.01em ~ 0.02em (약간 넓게) |
| 대문자(ALL CAPS) 텍스트 | 0.05em ~ 0.1em (넓게) |
| 라벨/배지 | 0.02em ~ 0.05em |

**원칙**: 큰 텍스트일수록 자간을 좁히고(tight), 작은 텍스트일수록 넓힌다(loose).

## 4. 행간 (Line Height)

| 요소 | 값 |
|---|---|
| 히어로 메인카피 | 1.1 ~ 1.2 |
| 섹션 타이틀 | 1.15 ~ 1.3 |
| 서브 헤딩 | 1.2 ~ 1.35 |
| 리드/서브카피 | 1.4 ~ 1.5 |
| 본문 (장문) | 1.5 ~ 1.7 |
| 캡션/설명 | 1.4 ~ 1.5 |
| 버튼 텍스트 | 1 ~ 1.2 |
| 네비게이션/UI 요소 | 1.2 ~ 1.3 |

**원칙**: 짧은 텍스트(헤딩)는 행간을 좁게, 긴 텍스트(본문)는 넓게. WCAG 접근성 최소 기준 1.5.

| 관계 | 간격 (px) | 비고 |
|---|---|---|
| 메인카피 ↔ 서브카피 | 12–16 | 밀접한 관계 |
| 서브카피 ↔ 설명 텍스트 | 16–24 | |
| 설명 텍스트 ↔ CTA 버튼 | 24–32 | |
| 헤딩 ↔ 본문 | 8–16 | 헤딩 아래 간격은 좁게 |
| 본문 단락 ↔ 단락 | 16–24 (font-size × 1.5~2) | |
| 섹션 ↔ 섹션 | 60–120 | |
| 섹션 내부 패딩 (상하) | 60–80 | Desktop 기준 |
| 섹션 내부 패딩 (좌우) | 24–40 또는 max-width 컨테이너 | |

## 6. 여백 시스템 (8pt Grid 스케일)

```
4px, 8px, 12px, 16px, 24px, 32px, 40px, 48px, 64px, 80px, 96px, 120px
```

대부분의 디자인 시스템에서 4px 또는 8px 기반 그리드를 사용한다.

## 7. 기타 핵심 규칙

- **폰트 사이즈 종류**: 한 페이지에 4~7개 이내로 제한
- **폰트 패밀리**: 최대 2~3개 (헤딩 1개 + 본문 1개 + 선택적 액센트 1개)
- **최적 줄 길이**: 한 줄당 45~75자 (영문), 한글은 25~40자
- **본문 최소 사이즈**: 16px 이상 (모바일 포함)
- **대비율**: 본문 최소 4.5:1, 큰 텍스트 3:1 (WCAG 기준)
- **히어로 메인카피**: 본문 사이즈의 최소 2~3배 이상
- **헤딩 위 간격 > 헤딩 아래 간격**: 헤딩은 아래 본문에 더 가깝게 배치
